===========================
GEEKY PUNKS CONVERTER
THE SOUND OF EVERYTHING
===========================

Manual

Convert Files
Import audio, optional Normalize, then Convert Files.
Output: WAV files saved to your output folder or downloads.

Kit Maker
Select up to 10 files, then Kit Maker.
Output: one folder H with samples numbered 0 to 9.

PWA Web App
PWA means you can install the web version like an app.
Open the web link, then install it.
Chrome: click Install App in the page, or use the browser menu → Install.
Safari on iPhone/iPad: Share → Add to Home Screen.
